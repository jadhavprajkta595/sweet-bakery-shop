# 🍰 Sweet Bakery Shop Website

A beautiful and responsive bakery website built using HTML and CSS.  
This project showcases bakery products, pricing, and ordering features with a modern UI design.

---

## 🧁 About The Project

Sweet Bakery Shop is designed to:

- Display delicious bakery products
- Show product prices
- Allow customers to place orders
- Provide contact information
- Work smoothly on desktop and mobile devices

---

## ✨ Features

- 🍞 Home Page
- 🛍️ Product Section
- 🧾 Order Page
- 📞 Contact Page
- 📱 Responsive Design
- 🎨 Clean and modern layout

---

## 🛠️ Built With

- CSS3
- JavaScript (if used)
